Class FetchFS

A simple filesystem backed by HTTP using the fetch API.

Listings objects look like the following:

{
  "home": {
    "jvilk": {
      "someFile.txt": null,
      "someDir": {
        // Empty directory
      }
    }
  }
}

This example has the folder /home/jvilk with subfile someFile.txt and subfolder someDir.
